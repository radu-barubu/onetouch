Record button

Nuance SpeechAnywhere automatically provides a button to start recording on web pages that are initialized for speech recognition:

Position

You can decide where on your web page you want the recording button to appear. If you do not configure the location the microphone button appears as the first element in the body of the web page.

Set the ID of the container that is to host the recording button; add this line to the NUSA_configure function: NUSA_container = "<container ID>";

Note: The container should allow enough space for the Install Silverlight button to be properly displayed. For more information, see: Microsoft Silverlight.

Appearance

You can customize the appearance of the recording button. If you do not customize the button the icon above is shown. Add these lines to the NUSA_configure function:

NUSA_recordButtonImage_RecordOn = "<URL to your image>";

NUSA_recordButtonImage_RecordOnHover = "<URL to your image>";

NUSA_recordButtonImage_RecordOff = "<URL to your image>";

NUSA_recordButtonImage_RecordOffHover = "<URL to your image>";

NUSA_recordButtonHeight = "<height of your image in pixels>";

NUSA_recordButtonWidth = "<width of your image in pixels>";

NUSA_recordButtonBackgroundColor = "<background color of your Silverlight control in hex code>";
