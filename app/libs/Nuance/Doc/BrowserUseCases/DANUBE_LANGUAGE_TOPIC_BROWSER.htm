Setting the language and topic

The language and topic properties are optional.

Language

You can specify the language to be used for speech recognition for each VUI form. Changing the language property after recording has been started for the first time will not have an effect.

IETF language tags are used. The language subtag is mandatory and the region subtag is optional (for example, en-GB for British English and en-US for US English). Other subtags are not supported.

If your web page sets the relevant language using the lang or xml:lang attribute it is not necessary to configure the language here. If you configure a different language here, this configuration overrides the web page setting.

Topic

Based on the medical specialties your users work in, you can specify the topic (medical specialty) for each VUI form. Changing the topic property after recording has been started for the first time will not have an effect.

The following topics are supported:

  • NUSA_topicGeneralMedicine - this is the default setting; if you do not specify a topic this topic is used.
  • NUSA_topicInternalMedicine
  • NUSA_topicSurgery
  • NUSA_topicMentalHealth
  • NUSA_topicNeurology
  • NUSA_topicCardiology

Procedure

In the NUSA_configure function, set the variable NUSA_language as required.

In the NUSA_configure function, set the variable NUSA_topic as required.

Note: These variables must be set before recording is started for the first time.
