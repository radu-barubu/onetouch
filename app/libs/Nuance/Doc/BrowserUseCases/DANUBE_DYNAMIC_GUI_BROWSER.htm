Dynamically modified controls

If you dynamically change the set of controls on your web page (for example, your application adds or removes specific text fields depending on the user's diagnosis of the patient) then the VUI form must be synchronized to recognize these changes.

Procedure

When the GUI view is changed after the VUI form has been created by Nuance SpeechAnywhere, call the NUSA_reinitializeVuiForm() method. The VUI form available to the user is adjusted the next time the user clicks the Record button.
