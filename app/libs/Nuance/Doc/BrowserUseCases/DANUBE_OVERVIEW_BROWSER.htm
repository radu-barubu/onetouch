Remarks

  • The angled brackets (< >) surrounding code examples indicate a placeholder. When you fill in real data make sure you do not include the angled brackets.
  • You must speech-enable every web page in your browser application that you want the user to be able to use with speech recognition.
  • When a user is working with your speech-enabled application and Nuance SpeechAnywhere cannot connect to the speech recognition server (for example, the Internet connection is lost) recording is stopped immediately. The feedback bubble no longer appears on screen. When the connection is restored the user can start again as normal.
  • User names are case sensitive.
