Technical restrictions

  • Silverlight 4.0 must be supported by your web browser.
  • Frame sets are not supported: only one page in a frame set can be speech-enabled.
  • By default the operating system's default audio device will be used; optionally, you can configure another audio device in the Silverlight configuration dialog box.
  • The maximum length of a text control is 50 kB.
