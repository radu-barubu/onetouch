Microphones

On Windows operating systems any microphone which can create 16-bit and at least 16 kHz audio data is supported.

On MacOS any microphone which can create 16-bit and at least 16 kHz audio data is supported.

Make sure to test the microphones that your users will be using with your speech-enabled web pages before you deploy your application.

If your users are using microphones that do not create audio data of good enough quality for speech recognition a message box is displayed; they can use your application but not with speech recognition.
