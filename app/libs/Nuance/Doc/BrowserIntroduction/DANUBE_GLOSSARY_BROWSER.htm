Glossary

A

Android edition Edition of Nuance SpeechAnywhere designed for Android devices; for example, Samsung Galaxy Tab.

audio device A device which is connected to a computer to aid recording and/or playback of sound; for example, a microphone.

B

Browser edition Edition of Nuance SpeechAnywhere designed for web browser applications.

C

command See: voice command.

control device A microphone (such as Philips SpeechMike or Nuance PowerMic) or foot pedal that provides (configurable) buttons for starting/stopping recording, etc.

correction Editing a text to replace incorrectly recognized words or phrases.

D

E

 

F

feedback bubble A GUI element providing the end user with visual feedback on recording. For more information, see: GUI elements.

field An end user text control in your application that can be speech-enabled. When the end user speaks, the recognized text is written into the field with the speech focus.

field navigation Moving between text controls using voice commands or by clicking.

frontend recognition The recorded text is recognized in real time on the GUI. Users can correct recognition errors immediately after or during a recording session. See also: speech recognition.

G

 

H

 

I

Insert mode Recording mode which inserts new speech at the current cursor position in the sound file but does not overwrite existing parts of the sound file.

iOS edition Edition of Nuance SpeechAnywhere designed for iOS devices; for example, iPhone, iPad.

J

 

K

KPI (Key Performance Indicator) Data about the Nuance SpeechAnywhere features that are used by the end user and how they are used. This data is collected, analyzed and used to tailor the product to real-life use cases.

L

license See: usage-based license.

M

medical specialty See: topic.

N

.NET edition Edition of Nuance SpeechAnywhere designed for .NET desktop applications.

Nuance Order Desk Service for ordering licenses.

Nuance partner Company who integrates Nuance SpeechAnywhere into its application.

O

Overwrite mode Recording mode which places new speech at the current cursor position in the sound file and overwriting existing speech.

P

partner See: Nuance partner.

Q

 

R

recognition result All sounds recognized by the speech recognition engine, including the dictated speech (which is converted into text), redundant speech and background noise (which are ignored), as well as voice commands (which are executed).

S

scratch that Command that deletes the most recently dictated text up to the last punctuation (comma, period , colon, paragraph, new line, etc.). The maximum length of the text that can be deleted using the scratch that command is approximately 3 minutes or 200 characters.

speech-enabled control GUI text control (field) that can accept spoken input and convert the sound to text.

speech focus Refers to the active speech-enabled text control (field) that will accept spoken input and display the recognized text.

speech recognition Computerized recognition of spoken words. Speech is transformed into text, this means written out or interpreted as voice commands to which the program responds with the requested action. See also: frontend recognition.

speech recognition language The language used for speech recognition.

T

topic The medical specialty end users work in.

U

usage-based license A license scheme based on the amount of sound data processed by speech recognition.

V

VA level Voice-activated recording level, also referred to as silence detection level, voice sensitivity threshold or VOR level. The VA level is a threshold level of the recording volume which allows the recording application to distinguish between speech (which should be recorded), and silence or background noise (which should not be recorded). Recording stops automatically when the speech input volume is below the VA level.

voice command A spoken word or phrase that initiates an action in an application.

VUI Voice User Interface

VUI bundle A custom set of voice commands that usually correspond to fields in a form. It can be seen as the VUI equivalent of a GUI view.
