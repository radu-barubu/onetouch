Introduction

With the release of Nuance SpeechAnywhere Release 1 the beta 2 version is no longer supported and the server has been switched to the new version.

For more information on all the features, browse this SDK documentation or see the end user documentation.

Integration changes necessary since the beta 2 release

To conform to HTML5 conventions the following changes were made:

  • The attribute NUSAConceptName has been renamed to data-nusa-concept-name.
  • The attribute NUSAIgnore has been renamed to data-nusa-enabled.

Known issues in Nuance SpeechAnywhere Release 1

  • The Nuance flame in speech-enabled controls is covered when the control has a scroll bar in the following browsers: Microsoft Internet Explorer 8, Google Chrome 9, Mozilla Firefox 3, Apple Safari 5.
  • If the connection to the speech recognition server is lost the current session cannot be resumed when the connection becomes available again.
  • On Mac OS no message is displayed to the user if the microphone they are using is not supported.
  • All text controls that are speech-enabled display the Nuance flame. The Nuance flame is still displayed even if speech recognition is not available.
