Welcome to Nuance SpeechAnywhere (Browser edition) help

Nuance SpeechAnywhere (Browser edition) enables you to add speech recognition functionality to your web browser applications. We have designed Nuance SpeechAnywhere to be easy for you to integrate and for end users to use.

The service provided by Nuance SpeechAnywhere is guaranteed to be secure and usable wherever the user is working.

When your web browser application is speech-enabled your users can use a variety of our features. For information on the speech recognition features available, what the user sees on screen and how to use the features, see: End user documentation. For information on general concepts, see: Glossary.

Organization of this documentation

This documentation contains the following sections:

  • Setting up Nuance SpeechAnywhere (Browser edition) - describes the package that we deliver.
  • Getting started - gives an overview of the steps involved in speech-enabling your application.
  • Nuance SpeechAnywhere design - describes how the product should be used.
  • Nuance SpeechAnywhere architecture - contains a system diagram.
  • Release Notes - lists changes to the product since the previous version.
  • Use cases - overview - leads you through speech-enabling your application.
  • On-premise deployment - describes how to configure the server/client connection if your system is deployed on-premise.
  • Microphones - provides information on supported microphones.
  • Technical restrictions - provides information that you should be aware of when integrating our package.
  • Microsoft Silverlight - provides information for the IT administrator who is deploying your application in the hospital system.
  • Acknowledgements - references the software of other companies that is included in Nuance SpeechAnywhere.
