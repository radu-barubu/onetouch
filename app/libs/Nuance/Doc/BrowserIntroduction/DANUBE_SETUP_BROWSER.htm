Setting up Nuance SpeechAnywhere (Browser edition)

Requirements

  • One of the following web browsers:

Mozilla Firefox 3.6, 4.0

Microsoft Internet Explorer 7, 8, 9

Google Chrome 8, 9, 10, 11

Apple Safari 5

  • Microsoft Silverlight 4.0. For more information, see: Microsoft Silverlight.
  • Entrust CA 2048 certificate installed on every client

To download it, go here: https://www.entrust.net/downloads/binary/entrust_2048_ca.cer

  • Cookies must be enabled in the browser in order for the information from the license and Partner GUIDs to be accessed; for more information, see: Speech-enabling your web browser application.
  • Java script must be enabled in the browser; for more information, see: Speech-enabling your web browser application.

Package contents

  • Nuance SpeechAnywhere contains:

Sample pages

SDK documentation

  • Samples contain:

Web pages that demonstrate integration of Nuance SpeechAnywhere.

Important: Sample pages and your application must be hosted on a web server. If you run them in a browser from a file Nuance SpeechAnywhere will not work.
